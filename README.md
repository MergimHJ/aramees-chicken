\# 🍗 Aramees Chicken



Site de commande en ligne - Cuisine libanaise et araméenne



\## 🚀 Fonctionnalités



\- ✅ Menu dynamique

\- ✅ Panier d'achat

\- ✅ Commande en ligne

\- ✅ Panel admin

\- ✅ Gestion produits

\- ✅ Gestion commandes



\## 🔧 Installation



1\. Clonez le dépôt

2\. Importez le SQL dans votre BDD

3\. Copiez `config.example.php` en `config.php`

4\. Modifiez `config.php` avec vos identifiants
